Présentation du projet

La Tanière Fertile est une ferme expérimentale connectée, développée par des étudiants de l’IUT Réseaux & Télécoms. Ce site permet au fermier et à ses acolytes de :

  • 📊 Surveiller les données en temps réel via Grafana
  • 💡 Contrôler les équipements à distance (effaroucheur, lampe, clim...)
  • 📍 Localiser les dispositifs sur une carte
  • 📲 Recevoir des alertes via un bot Telegram

Explorez les différentes sections via la navigation ci-dessus.

🧠 Comment fonctionne notre système

Notre ferme connectée repose sur une architecture moderne composée des briques suivantes :

  • Capteurs LoRaWAN : transmettent les données environnementales (bruit, température...) sur de longues distances avec très faible consommation.
  • MQTT : protocole léger utilisé pour publier les données vers nos serveurs.
  • Telegraf : agent chargé de collecter, transformer et envoyer les données vers InfluxDB.
  • InfluxDB : base de données spécialisée dans les séries temporelles, stocke toutes les mesures de capteurs.
  • Grafana : interface visuelle permettant de créer des tableaux de bord dynamiques pour le suivi des capteurs.
  • Telegram Bot : envoie automatiquement des alertes en cas d’anomalie détectée (bruit fort, panne...)

Toutes ces briques communiquent entre elles pour permettre au fermier une gestion simple, efficace et accessible depuis un navigateur.
